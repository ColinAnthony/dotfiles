# dotfiles

## Usage
1. Clone this repo.
```
$ git clone https://github.com/mom0tomo/dotfiles.git
$ cd dotfiles
```

2. Set symbolic link.
```
$ ln -s $PWD/.gitconfig ~/.gitconfig

// for fish
$ ln -s $PWD/fish/ ~/.config/

// for vim
$ ln -s $PWD/.vimrc ~/.vimrc
$ ln -s $PWD/.vim/ ~/

// for vim GUI 
$ ln -s $PWD/.gvimrc ~/.gvimrc
```

### for neovim
```
$ ln -s $PWD/.vim/colors/ ~/.config/
```
